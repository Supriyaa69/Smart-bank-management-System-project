# Smart-bank-management-System-project
Full-stack Smart Bank Management System developed using Spring Boot, MySQL, HTML, CSS, and JavaScript with features including authentication, account management, fund transfer, transaction history, loan processing, and admin approval workflow.

---

##Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- HTML
- CSS
- JavaScript
- REST APIs

---

##Features

###User Module
- User Registration & Login
- Account Creation
- Deposit Money
- Fund Transfer
- Transaction History
- Loan Application

###Admin Module
- View Loan Requests
- Approve / Reject Loans

---

##Architecture

The project follows layered architecture using:

- Controller Layer
- Service Layer
- Repository Layer
- DTO Layer
- Mapper Layer

---

##Banking Functionalities

- Transaction Management using `@Transactional`
- Real-time balance updates
- Role-based workflow handling
- REST API integration
- Frontend-backend communication using Fetch API

---

##Frontend Modules

- Login Page
- Registration Page
- Dashboard
- Deposit Module
- Transfer Module
- Transaction History
- Loan Management
- Admin Approval Panel

---

##How to Run

1. Clone the repository
2. Configure MySQL database in `application.properties`
3. Run the Spring Boot application
4. Open browser:
   ```
   http://localhost:8080/login.html
   ```
